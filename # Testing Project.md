# Testing Project

A starter testing project for JavaScript (ES6), Vue.js, Quasar Framework, and Apache Cordova.

## Prerequisites
- Node.js (LTS) and npm
- Vue CLI / Quasar CLI
- Cordova CLI
- Git (optional)

## Setup
1. Clone or download the repo.
2. Install dependencies:
    ```bash
    npm install
    ```
3. If using Quasar:
    ```bash
    npm install -g @quasar/cli
    ```
4. If using Cordova:
    ```bash
    npm install -g cordova
    ```

## Scripts
- Run unit tests:
  ```bash
  npm test
  ```
- Run lint:
  ```bash
  npm run lint
  ```
- Run dev server (Vue/Quasar):
  ```bash
  npm run dev
  ```
- Build (Vue/Quasar):
  ```bash
  npm run build
  ```
- Cordova platform add (example):
  ```bash
  cordova platform add android
  ```
